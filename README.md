# StatBot Pro

AI-powered CSV Data Analyst Agent built using:

- FastAPI
- Streamlit
- Gemini AI
- LangChain
- Pandas
- Matplotlib

## Features
- CSV upload
- Natural language dataset analysis
- AI-generated insights
- Automatic charts and visualizations
- Interactive dashboard

## Run Project

```bash
uvicorn app.main:app --reload

Frontend:
streamlit run frontend/app.py
